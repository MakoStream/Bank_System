Detailed Description

Helper functions for string, number, and configuration handling.

This header defines utility functions used throughout the bank system, including string validation, string splitting and trimming, configuration file reading and writing, and timestamp generation.

Note
Required for classes and functions like NewAccountCommand, UserListCommand, AccountListCommand, UserInfoCommand, RegisterUserCommand, mainProcess, and Command::execute().
