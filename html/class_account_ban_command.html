Detailed Description

Handles banning of a bank account.

The AccountBanCommand class performs:

  • Validates IBAN and optional reason
  • Checks if the account exists
  • Bans the account and updates storage
  • Sets hash[0] = 1 on success
  • Logs all actions via logEye
    Note
    Requires: <string>, Account, handleInfo, logEye
    Syntax: account_ban <IBAN> [reason]
