\doxysubsection{Detailed Description}
Main class for session, user, transaction, and configuration management. 

The \doxylink{classmain_process}{main\+Process} class handles\+:
\begin{DoxyItemize}
\item \doxylink{class_user}{User} session management (create, login, logout, generate authentication keys)
\item Database operations (file paths, incrementing PAN, IBAN, transaction IDs)
\item Transaction handling and logging via Audit
\item Debug mode management
\item Reading and writing configuration files \begin{DoxyNote}{Note}
Requires\+: \doxylink{_bank___system_2basic__functions_8cpp_a248127881de11156f58e4dd2fd879f6a}{read\+Config()}, \doxylink{_bank___system_2basic__functions_8cpp_a006014cced648ca7880d24c53223bdd4}{write\+Config()}, $<$vector$>$, $<$string$>$, $<$map$>$, \doxylink{class_logger}{Logger}, \doxylink{structsession_construct}{session\+Construct}, \doxylink{struct_session}{Session}, \doxylink{class_user}{User}, account @required For all core program operations (\doxylink{class_commands_manager}{Commands\+Manager}, Handle\+Client, \doxylink{classuser_login_command}{user\+Login\+Command}, transaction\+\_\+request, etc.) @used For\+:
\end{DoxyNote}

\item Processing client sessions
\item \doxylink{class_user}{User} login/logout
\item Creating and approving transactions
\item Configuration file management 
\end{DoxyItemize}
