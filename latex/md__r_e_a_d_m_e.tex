\hypertarget{md__r_e_a_d_m_e_autotoc_md9}{}\doxysection{\texorpdfstring{Overview}{Overview}}\label{md__r_e_a_d_m_e_autotoc_md9}
Mini Bank System is an educational project that simulates the operation of a banking system with limited functionality. It demonstrates object-\/oriented programming principles and client-\/server interaction using local Pipes as a communication mechanism.\hypertarget{md__r_e_a_d_m_e_autotoc_md10}{}\doxysection{\texorpdfstring{Objective}{Objective}}\label{md__r_e_a_d_m_e_autotoc_md10}
The main goal of the project is to create a simplified banking environment that allows users and administrators to interact with a local server and perform basic banking operations.\hypertarget{md__r_e_a_d_m_e_autotoc_md11}{}\doxysection{\texorpdfstring{Project Structure}{Project Structure}}\label{md__r_e_a_d_m_e_autotoc_md11}
The system consists of three main applications\+:
\begin{DoxyEnumerate}
\item User Client
\item Admin Client
\item Server Client
\end{DoxyEnumerate}

The User and Admin clients communicate with the Server Client through local Pipes (temporary files). A full network server implementation is planned for the extended stage of the project.\hypertarget{md__r_e_a_d_m_e_autotoc_md12}{}\doxysection{\texorpdfstring{Main Features (Stage 1)}{Main Features (Stage 1)}}\label{md__r_e_a_d_m_e_autotoc_md12}

\begin{DoxyEnumerate}
\item Server-\/side database implemented using binary files.
\item User management system (login, logout, profile update).
\item Bank account management (creation, deposits, withdrawals, transfers).
\item Support for different account types\+: default, credit, deposit.
\item Logging system for all database and banking operations.
\item Admin client for managing users, accounts, and transactions.
\item Simple text-\/based interface for User and Admin clients.
\end{DoxyEnumerate}\hypertarget{md__r_e_a_d_m_e_autotoc_md13}{}\doxysection{\texorpdfstring{Future Development (Stage 2)}{Future Development (Stage 2)}}\label{md__r_e_a_d_m_e_autotoc_md13}

\begin{DoxyEnumerate}
\item Financial reporting in CSV format.
\item Automated data backups and security improvements.
\item Full server implementation using sockets.
\end{DoxyEnumerate}\hypertarget{md__r_e_a_d_m_e_autotoc_md14}{}\doxysection{\texorpdfstring{Author and Year}{Author and Year}}\label{md__r_e_a_d_m_e_autotoc_md14}
