Manages and executes all available commands. 

Registers all available command objects.

Stores all command objects in a vector and matches incoming command strings from client sessions to the appropriate \doxylink{class_command}{Command} object for execution.

The constructor adds instances of all concrete \doxylink{class_command}{Command} classes to the internal vector, including both standard and debug/test commands. \begin{DoxyNote}{Note}
Used internally when creating a \doxylink{class_commands_manager}{Commands\+Manager} object. 
\end{DoxyNote}

Constructor that registers all available commands. 

Executes a command based on the string received from a client.

Initializes the vector of commands, including debug and test commands. \begin{DoxyNote}{Note}

Parses the command string from session\+Data, finds the matching \doxylink{class_command}{Command} object, executes it, and clears the cmd field. If the command does not exist, sends a response to the client indicating an unknown command.
