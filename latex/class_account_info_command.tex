\doxysubsection{Detailed Description}
Retrieves detailed information about an account by card number. 

Validates card number, checks if account exists, retrieves IBAN, balance, currency, and owner information, logs all actions via log\+Eye. \begin{DoxyNote}{Note}
Requires\+: $<$string$>$, Account, \doxylink{structhandle_info}{handle\+Info}, log\+Eye
