\doxysubsection{Detailed Description}
Handles banning of a bank account. 

The \doxylink{class_account_ban_command}{Account\+Ban\+Command} class performs\+:
\begin{DoxyItemize}
\item Validates IBAN and optional reason
\item Checks if the account exists
\item Bans the account and updates storage
\item Sets hash\mbox{[}0\mbox{]} = 1 on success
\item Logs all actions via log\+Eye \begin{DoxyNote}{Note}
