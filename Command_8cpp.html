Sends a response message to the client.

Copies the given message into the first slot of sessionData.msg and writes the updated sessionData structure to the pipe associated with the handle.

Parameters
handleReference to handleInfo containing the pipe and sessionData.
msgThe message string to send to the client.
Note
Required headers: <string>, <cstring>, <windows.h>.
Used by Command and all derived command classes to send messages to clients.
